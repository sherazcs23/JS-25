Q1.What is JavaScript?
JavaScript is a scripting language used to create dynamic and interactive web pages.

Q2.Is it synchronous or asynchronous?
JavaScript is single-threaded and synchronous by default but supports asynchronous behavior via callbacks, promises, and async/await.

Q3.Single-line or multi-line programming language?
JavaScript supports both single-line and multi-line code.

Q4.JS runs on which engine?
JavaScript runs on engines like Google's V8 (Chrome, Node.js), SpiderMonkey (Firefox), and others.

Q4.Usage of console.log() in JavaScript:
console.log() is used to print messages to the browser console for debugging or displaying output.
example console.log("Hello world");
output // Hello world .
